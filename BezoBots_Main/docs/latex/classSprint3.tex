\doxysubsection{Detailed Description}
A ROS2 node for processing laser scan, odometry, and occupancy grid data to detect and map cylindrical objects. 

This node subscribes to the laser scan, occupancy grid, and odometry topics to detect cylinder-\/like objects (circles) which are 30cm in diameter in the project\textquotesingle{}s environment. Detected circles are then transformed into the map frame and visualized on the occupancy grid (with the position).
